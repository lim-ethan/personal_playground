# 02-05 정규 표현식 (Regular Expression)

텍스트 전처리에서 정규 표현식은 아주 유용한 도구입니다.  
이번에는 파이썬에서 지원하고 있는 `re` 모듈의 사용 방법과 NLTK를 통한 정규 표현식을 이용한 토큰화에 대해서 알아봅니다.

---

## 1. 정규 표현식 문법과 모듈 함수

파이썬에서는 정규 표현식 모듈 `re`를 지원하므로, 이를 이용하면 특정 규칙이 있는 텍스트 데이터를 빠르게 정제할 수 있습니다.  
아래는 정규 표현식을 위해 사용되는 특수 문자들과, `re` 모듈의 함수들에 대한 설명입니다.

---

### 1) 정규 표현식 문법 (특수 문자)

| 기호 | 설명 |
|------|------|
| `.` | 한 개의 임의의 문자 (단, 줄바꿈 문자 `\n` 제외) |
| `?` | 앞의 문자가 존재할 수도 있고 존재하지 않을 수도 있음 (0개 또는 1개) |
| `*` | 앞의 문자가 0개 이상 존재 가능 |
| `+` | 앞의 문자가 1개 이상 존재 |
| `^` | 뒤의 문자열이 **문자열의 시작**과 매치되어야 함 |
| `$` | 앞의 문자열이 **문자열의 끝**과 매치되어야 함 |
| `{숫자}` | 정확히 해당 숫자만큼 반복 |
| `{숫자1, 숫자2}` | 숫자1 이상 숫자2 이하만큼 반복 |
| `{숫자,}` | 숫자 이상 반복 |
| `[ ]` | 대괄호 안의 문자 중 하나와 매치 (예: `[a-z]`) |
| `[^문자]` | 해당 문자를 제외한 문자와 매치 |
| `|` | OR 연산자 (예: `A|B`) |

#### 문자 클래스 & 이스케이프 시퀀스

| 기호 | 설명 |
|------|------|
| `\\` | 역슬래시 문자 자체 |
| `\d` | 모든 숫자 (0–9) |
| `\D` | 숫자가 아닌 모든 문자 |
| `\s` | 공백 문자 (스페이스, 탭, 줄바꿈 등) |
| `\S` | 공백이 아닌 문자 |
| `\w` | 알파벳 또는 숫자 (언더스코어 포함) |
| `\W` | 알파벳이나 숫자가 아닌 문자 |

---

### 2) 정규 표현식 모듈 함수 (`re` 모듈)

| 함수 | 설명 |
|------|------|
| `re.compile(pattern)` | 정규 표현식을 컴파일하여 패턴 객체로 만듭니다. 자주 쓰는 패턴이면 미리 컴파일해 두면 성능상 유리합니다. |
| `re.search(pattern, string)` | 문자열 전체에서 패턴과 매치되는 부분이 있는지 검색. 매칭되면 `Match` 객체 리턴. |
| `re.match(pattern, string)` | 문자열의 시작부터 패턴이 매치되는지 확인. 문자열 중간에 패턴이 있어도 무시됨. |
| `re.split(pattern, string)` | 패턴을 기준으로 문자열을 분리하여 리스트로 반환. |
| `re.findall(pattern, string)` | 패턴과 매치되는 모든 문자열을 리스트로 반환. 매치되는 것이 없으면 빈 리스트 반환. |
| `re.finditer(pattern, string)` | 매치되는 모든 곳에 대해 `Match` 객체들의 이터레이터를 반환. |
| `re.sub(pattern, repl, string)` | 패턴과 매치되는 부분을 `repl` 문자열로 대체하여 반환. |

## 2. 정규 표현식 실습

아래는 주요 특수 문자를 사용하는 예제입니다.

---

### (1) `.`

정규 표현식이 `a.c`라면, a와 c 사이에 임의의 단일 문자가 오는 경우 매치됩니다.

```python
import re
r = re.compile("a.c")
print(r.search("kkk"))   # None
print(r.search("abc"))   # Match
```

### (2) ?

ab?c는 abc 또는 ac와 매치됩니다. (b가 있을 수도, 없을 수도)
```python
r = re.compile("ab?c")
print(r.search("abbc"))  # None
print(r.search("abc"))   # Match
print(r.search("ac"))    # Match
```
---
### (3) *

ab*c는 ac, abc, abbc, abbbc 등과 매치됩니다. (b가 0개 이상)
```python
r = re.compile("ab*c")
print(r.search("ac"))      # Match
print(r.search("abc"))     # Match
print(r.search("abbbc"))   # Match
```

---
### (4) +

ab+c는 abc, abbc, abbbc 등과 매치됩니다. (b가 최소 1개 이상)
```python
r = re.compile("ab+c")
print(r.search("ac"))      # None
print(r.search("abc"))     # Match
print(r.search("abbbc"))   # Match
```

---

### (5) ^

^ab는 문자열이 ab로 시작해야 매치됩니다.
```python
r = re.compile("^ab")
print(r.search("kkkabc"))  # None
print(r.search("abz"))     # Match
```

---
### (6) {숫자}

ab{2}c는 abbc만 매치됩니다.
```python
r = re.compile("ab{2}c")
print(r.search("abbc"))   # Match
```
---
### (7) {숫자1,숫자2}

ab{2,8}c는 b가 2개에서 8개 사이인 경우 매치됩니다.
```python
r = re.compile("ab{2,8}c")
print(r.search("abbbbbbc"))  # Match
```
---
### (8) {숫자,}

a{2,}bc는 a가 2개 이상 반복되고 그 뒤에 bc가 오는 경우 매치됩니다.
```python
r = re.compile("a{2,}bc")
print(r.search("aabc"))  # Match
```
---
### (9) [ ]

[abc]는 a, b, c 중 하나와 매치됩니다.
범위 사용 가능: [a-z], [A-Z], [0-9]
```python
r = re.compile("[a-z]")
print(r.search("A"))  # None
print(r.search("z"))  # Match
```
---
### (10) [^문자]

[^abc]는 a, b, c를 제외한 문자와 매치됩니다.
```python
r = re.compile("[^abc]")
print(r.search("a"))  # None
print(r.search("d"))  # Match
```

## 3. re 모듈 함수 실습

### (1) match() vs search()

match()는 문자열의 시작 부분만 확인,
search()는 문자열 전체에서 검색합니다.
```python
r = re.compile("ab.")
print(r.match("kkkabc"))  # None
print(r.search("kkkabc")) # Match
print(r.match("abckkk"))  # Match
```
---
### (2) re.split()

패턴을 기준으로 문자열을 분리하여 리스트로 반환합니다.
```python
import re
text = "사과 딸기 수박 메론 바나나"
print(re.split(" ", text))
# ['사과', '딸기', '수박', '메론', '바나나']
```
---
### (3) re.findall()
패턴과 매치되는 모든 부분 문자열을 리스트로 반환합니다.
```python
text = """이름 : 김철수
전화번호 : 010 - 1234 - 1234
나이 : 30
성별 : 남"""
print(re.findall("\d+", text))
# ['010', '1234', '1234', '30']
```
---

### (4) re.sub()

패턴과 매치되는 부분을 다른 문자열로 치환합니다.

```python
text = "Regular expression : A regular expression, regex or regexp ..."
result = re.sub('[^a-zA-Z]', ' ', text)
print(result)
# 알파벳이 아닌 문자는 공백으로 치환
```
---
## 4. 텍스트 전처리 예제
```python
text = """100 John    PROF
101 James   STUD
102 Mac   STUD"""

print(re.split('\s+', text))
# ['100', 'John', 'PROF', '101', 'James', 'STUD', '102', 'Mac', 'STUD']

print(re.findall('\d+', text))
# ['100', '101', '102']

print(re.findall('[A-Z]{4}', text))
# ['PROF', 'STUD', 'STUD']

print(re.findall('[A-Z][a-z]+', text))
# ['John', 'James', 'Mac']
```

## 5. 정규 표현식을 이용한 토큰화 (NLTK)

NLTK에서는 정규 표현식을 사용한 토큰화를 지원합니다.
```python
from nltk.tokenize import RegexpTokenizer

text = "Don't be fooled by the dark sounding name, Mr. Jone's Orphanage is as cheery ..."

tokenizer1 = RegexpTokenizer(r"\w+")
print(tokenizer1.tokenize(text))
# ['Don', 't', 'be', 'fooled', 'by', 'the', ...]

tokenizer2 = RegexpTokenizer(r"\s+", gaps=True)
print(tokenizer2.tokenize(text))
# ["Don't", 'be', 'fooled', 'by', 'the', 'dark', ...]
```
- \w+ : 알파벳/숫자가 연속된 문자열을 토큰으로 인식
- gaps=True : 해당 패턴을 구분자로 사용
